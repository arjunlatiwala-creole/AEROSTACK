# Agentic DevOps Showcase

Welcome to the **Agentic DevOps Showcase** repository. This project serves as a live, intentionally flawed environment designed to present advanced Agentic DevOps capabilities in a realistic enterprise setting.

This repository integrates AWS infrastructure, Kubernetes orchestration, Docker containers, and CI/CD pipelines to provide AI agents with a comprehensive playground for root cause analysis, security remediation, automated provisioning, and more.

## 🏗️ Architecture Overview

The showcase is built on a modern, scalable architecture designed to mimic real-world enterprise constraints:

- **Frontend**: A React-based Progressive Web App (PWA) configured with Vite and Docker.
- **Tools API**: Node.js microservices handling core business logic, originally built on AWS serverless but now standardized on Terraform.
- **Infrastructure**: Standardized AWS components (VPC, EKS, RDS PostgreSQL) provisioned via Terraform.
- **Observability**: A complete `kube-prometheus-stack` providing metrics and visualization via Grafana.

## 📁 Repository Structure

```text
.
├── pwa-frontend/           # React frontend application
├── tools-api/              # Node.js backend services
├── infra/                  # Terraform Infrastructure as Code (EKS, VPC, RDS)
├── k8s/                    # Kubernetes manifests and Helm charts
├── .github/workflows/      # CI/CD pipelines (Trivy, Checkov, Deployment)
├── docs/                   # Documentation generated by agents
└── runbooks/               # Auto-generated cutover runbooks
```

## 🧪 Presentation Capabilities

This environment is pre-configured with intentional "faults" that allow you to present various automated workflows. The repository contains complex interactions, missing alerts, flaky tests, and infrastructure drift, making it the perfect sandbox for AI agents to diagnose and fix real-world problems.

## 🔒 Security Note

**DO NOT DEPLOY THIS REPOSITORY TO A PRODUCTION ENVIRONMENT UNMODIFIED.** 
It contains intentional security vulnerabilities, exposed ports, and non-compliant configurations specifically designed to be caught and remediated by AI agents during presentations.
